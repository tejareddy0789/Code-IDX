# React + Vite

This project was created using the Vite build tool, which provides a fast and optimized development environment for modern web applications. The project is set up with React, enabling the creation of dynamic and interactive user interfaces.

## Technologies Used

- **React**: A JavaScript library for building user interfaces.
- **Vite**: A modern build tool that offers fast development and optimized builds.
- **ESLint**: A tool for identifying and fixing problems in JavaScript code.
- **Babel**: Used for transforming modern JavaScript into a format compatible with older environments.
- **SWC**: A super-fast compiler for JavaScript used for Fast Refresh during development.

## Features

- Hot Module Replacement (HMR) for a seamless development experience.
- Minimal setup to get started quickly with React and Vite.
- Expandable ESLint configuration for maintaining code quality.
